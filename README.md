# Cat vs Dog Image Classification

This project is a CNN-based image classifier built using PyTorch to classify images of cats and dogs.  
It also includes a Streamlit web application for real-time predictions.

## Features
- CNN model using PyTorch
- Train & test pipeline
- Accuracy and confusion matrix
- Streamlit web app

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train.py

3. Evaluate:
   python evaluate.py

4. Run web app:
   streamlit run app.py
